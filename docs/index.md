## 레이어별 문서 매핑

각 레이어 챕터는 해당 레이어의 파일 범위, 설계 의도, 코드 패턴, 확장 가이드를 포함한다.

```
core/       → 4장 (State, Schema, Model Factory)
memory/     → 5장 (체크포인터, 영속 저장소)
prompts/    → 6장 (템플릿, 동적 빌더)
tools/      → 7장 (베이스, 레지스트리, 개별 도구)
nodes/      → 8장 (추론, 실행, 라우팅)
graphs/     → 9장 (빌더, 서브그래프, 메인)
interfaces/ → 10장 (REST API, 스트리밍)
config/     → 11장 (환경 변수, YAML 구성)
```
